# arduino-power-shield
A custom Arduino Uno Shield designed with **Altium Designer** as a capstone project for my *Altium Education PCB Design Course.*

The board implements a step-down voltage regulator to power a bank of LEDs while also supplying regulated power directly to the Arduino, bypassing its onboard regulator.

## Overview
| Parameter | Value |
|---|---|
|Form Factor| Arduino Uno Shield|
|Tool| Altium Designer|
|Design Type| Analog Power + Digital Interface|
| Outputs| LED Power Rail + Arduino 5V Supply|
